# ShadowForge Core

Kernel-level resilient infrastructure toolkit for high-adversity environments.

## Overview
ShadowForge provides tools for building decentralized and survivable systems.

## Features
- Polymorphic evasion mechanisms
- System persistence capabilities
- Mesh networking components

## Quick Start
```bash
git clone https://github.com/LizaVice/shadowforge-core.git
cd shadowforge-core
cargo build --release
./target/release/shadowforge persist
